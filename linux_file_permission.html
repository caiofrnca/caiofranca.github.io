For example, the string `drwxrwxrwx` represents a directory where the owner, group, and others have full permissions: - The **1st character** indicates the file type (`d` for directory, `-` for a regular file). - The **2nd-4th characters** represent the user’s permissions (read `r`, write `w`, execute `x`). - The **5th-7th characters** indicate the group’s permissions. - The **8th-10th characters** represent permissions for others.
